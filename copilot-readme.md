# ⚔️ ML Squad Hub

A Mobile Legends community platform for finding teammates, forming squads, and organizing scrims. Built for competitive players who want to connect, team up, and improve their gameplay experience.

---

## 🟢 Overview

ML Squad Hub is a web application that allows Mobile Legends players to:

- Create a player profile
- Find teams or members (LFT / LFM system)
- Organize and join scrims
- Chat with the global community
- Connect using ML ID, rank, and roles

The platform is designed with a **green and black esports theme** to match competitive gaming aesthetics.

---

## 🚀 Features

### 👤 User System

- Username & password authentication
- Player profile creation
- ML account details:
  - ML ID
  - Role (Tank, Jungle, Mid, Gold, EXP, Roam)
  - Current Rank
  - Highest Rank
  - Facebook link
  - Profile image

---

### 🧩 Team Finder (LFT / LFM)

- Post “Looking for Team” (LFT)
- Post “Looking for Members” (LFM)
- Role-based filtering
- Rank-based filtering
- Join or apply to teams

---

### ⚔️ Scrim System

- Create scrim requests
- Set match schedule
- Define match format (BO1, BO3, BO5)
- Add 5-member team lineup (ML IDs)
- View and accept scrim challenges

---

### 💬 Global Chat

- Real-time global chat lobby
- Connect with other players
- Find teams or scrims quickly

---

## 🧱 Tech Stack

### Frontend

- Next.js
- TypeScript
- Tailwind CSS (green/black esports theme)

### Backend

- Express (or Next API routes)
- supabase

- WebSockets (for chat)
- Prisma ORM
